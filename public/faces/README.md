# Reference face photos

Optional: add two front-facing photos for automatic setup:

| File | Config name (default) |
|------|------------------------|
| `person1.jpg` | Alice |
| `person2.jpg` | Bob |

If these files are missing, open the **Face Recognition** tab and use **Capture Alice** / **Capture Bob** to register from the webcam instead.

Requirements for photo files:
- Clear, well-lit, front-facing photo
- One face only per image
- JPG format

Update display names in `src/config/faceProfiles.ts` if needed.
